简单灵活的在游戏中的每秒计数帧，先进的功能，取得了让你监视你的游戏的统计权的目标设备上。



可见，通过3个计数器统计：


• FPS Counter


• Memory Counter


• Device Information Counter



有很多自定义设置。



看到它在行动在WebGL 演示的.



查看每个柜台及以下的其他一些额外的信息，更多的细节。



== FPS Counter 特征 ==


帧每秒，很明显。


帧的时间以毫秒为单位。


平均FPS可定制的样品。


最大和最小FPS与启动延迟。


在3个自定义的时间间隔自定义颜色。



== Memory Counter 特征 ==


总的专用内存。


分配的内存。


单内存。


可选精确的方式更加准确。


自定义颜色。



== Device Information Counter 特征 ==


自定义颜色。


CPU: 型号，核心数。


GPU 第1行: 模型，API版本。


GPU 第2行: 着色器模型，填充率和显存。


RAM: 总量上的当前设备。


银幕: 分辨率，刷新率，窗口大小，DPI。



== 常见特性 ==


可选的背景（无形）模式。


全局热键启用/禁用插件。


力的FPS可进行模拟低端设备。


保持连接的选项中生存场景切换。


灵活的计数器锚固。


聪明的“脏”更新系统。


独立的更新间隔计数器。


自定义颜色为每个计数器。


自定义比例，字体，间距，像素偏移量。


而更多的（见截图）！
